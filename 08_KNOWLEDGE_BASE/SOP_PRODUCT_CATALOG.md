## ПРОЦЕДУРА 1: ДОБАВЛЕНИЕ НОВОГО ТОВАРА

### Шаг 1 — Скопировать шаблон
Скопировать `TEMPLATE_PRODUCT_CARD.md` в папку `products/`.  
Назвать файл по артикулу: `[БРЕНД]-[КОД]-[НОМЕР].md`

**Пример:** `YA-JEANS-001.md`, `DROPS-LIMA-003.md`

### Шаг 2 — Заполнить обязательные поля
Минимально необходимые поля для публикации:

| Поле | Обязательно | Примечание |
|------|-------------|------------|
| Название | Да | Полное коммерческое название |
| Бренд | Да | Точное название бренда |
| Категория | Да | Из утверждённого списка категорий |
| Артикул | Да | Уникальный, без пробелов |
| Состав | Да | В % (пример: 55% хлопок, 45% акрил) |
| Цена (тенге) | Да | Актуальная розничная цена |
| Остаток (шт) | Да | Реальный остаток на складе |
| Описание для покупателя | Да | Минимум 50 слов |
| Ключевые слова | Да | Минимум 5 фраз |
| SEO описание | Желательно | 150–300 слов |
| Метраж / Вес / Цвет | Если применимо | Для пряжи — обязательно |

### Шаг 3 — Проверить уникальность артикула
Перед сохранением убедиться, что артикул не используется в другой карточке.

### Шаг 4 — Установить статус
- Новая карточка → `[x] Черновик`
- После проверки → `[x] Готов`
- После публикации на сайте/маркетплейсе → `[x] Опубликован`

### Шаг 5 — Обновить дату
Заполнить поля `Дата добавления` и `Дата обновления`.

---

## ПРОЦЕДУРА 2: ОБНОВЛЕНИЕ СУЩЕСТВУЮЩЕГО ТОВАРА

1. Открыть карточку товара в `products/`
2. Внести изменения в нужные поля
3. Обновить поле `Дата обновления`
4. Если изменилась цена или остаток — синхронизировать с сайтом и маркетплейсами вручную (до автоматизации)

**Важно:** никогда не менять `Артикул` у существующего товара — это нарушит ссылки в других системах.

---

## ПРОЦЕДУРА 3: СНЯТИЕ ТОВАРА С ПРОДАЖИ

1. Открыть карточку товара
2. Установить `Остаток (шт): 0`
3. Изменить статус на `[ ] Черновик`
4. Добавить в конец файла строку: `> Снят с продажи: [дата] — [причина]`
5. НЕ удалять файл — перенести в `99_ARCHIVE/products/`

## ПРАВИЛА ИМЕНОВАНИЯ

| Элемент | Правило | Пример |
|---------|---------|--------|
| Артикул | Бренд-серия-номер, латиница, заглавные | `YA-JEANS-001` |
| Имя файла | Совпадает с артикулом + `.md` | `YA-JEANS-001.md` |
| Название товара | Бренд + серия + уточнение | `Пряжа YarnArt Jeans` |
| Цвет | Название + код в скобках | `Белый (01)` |

## КОНТРОЛЬ КАЧЕСТВА

Перед переводом карточки в статус `Готов` проверить:

- [ ] Артикул уникален
- [ ] Все обязательные поля заполнены
- [ ] Цена актуальна
- [ ] Остаток соответствует складу
- [ ] Описание для покупателя — минимум 50 слов, без орфографических ошибок
- [ ] Ключевые слова включают название бренда и категорию
- [ ] Статус выставлен корректно
