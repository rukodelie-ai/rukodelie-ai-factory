## МИССИЯ

Построить полностью автономную AI-фабрику, которая управляет, продаёт, продвигает и масштабирует бизнес Rukodelie.kz — превращая ручной труд в системный, измеримый и масштабируемый поток ценности.

**Ключевой принцип:** каждый агент — это автономный сотрудник с чёткой зоной ответственности, метриками и протоколами взаимодействия.

## РОЛИ АГЕНТОВ

### 1. AI Project Manager (`00_PROJECT_MANAGER`)
**Приоритет:** P0 — критический  
**Ответственность:** стратегическое планирование, распределение задач между агентами, контроль исполнения, еженедельные отчёты владельцу, приоритизация бэклога.  
**Входы:** цели владельца, метрики всех агентов, события в проекте.  
**Выходы:** TASKS.md, ROADMAP.md, статус-отчёты, задачи агентам.  
**KPI:** % задач выполнено в срок, coverage агентов активными задачами.

---

### 2. AI Seller (`01_AI_SELLER`)
**Приоритет:** P0 — критический  
**Ответственность:** обработка входящих обращений, квалификация лидов, презентация товаров, работа с возражениями, оформление заказов, up-sell / cross-sell, возвраты.  
**Входы:** запросы клиентов (Telegram, сайт), данные из Product Database.  
**Выходы:** оформленные заказы, лог разговоров, аналитика конверсии.  
**KPI:** конверсия лид→заказ, средний чек, NPS клиентов.

---

### 3. Product Database (`02_PRODUCT_DATABASE`)
**Приоритет:** P1 — высокий  
**Ответственность:** ведение каталога товаров, актуализация цен и остатков, генерация карточек товаров, синхронизация с торговыми площадками.  
**Входы:** данные поставщиков, решения владельца по ценообразованию.  
**Выходы:** структурированный каталог (JSON/CSV), карточки для маркетплейсов, API для других агентов.  
**KPI:** актуальность данных (% карточек без ошибок), скорость обновления.

---

### 4. Telegram Bot (`03_TELEGRAM`)
**Приоритет:** P1 — высокий  
**Ответственность:** точка входа для клиентов и владельца, маршрутизация запросов к AI Seller, рассылки, уведомления о заказах, внутренние алерты фабрики.  
**Входы:** сообщения пользователей, события от других агентов.  
**Выходы:** ответы клиентам, уведомления, команды агентам.  
**KPI:** время ответа, % автоматически обработанных запросов.

---

### 5. SEO Agent (`04_SEO_AGENT`)
**Приоритет:** P2 — средний  
**Ответственность:** семантическое ядро, SEO-аудит сайта, генерация оптимизированного контента (описания, статьи, метатеги), мониторинг позиций.  
**Входы:** каталог товаров, анализ конкурентов, поисковые тренды.  
**Выходы:** SEO-тексты, рекомендации по структуре сайта, отчёты по позициям.  
**KPI:** органический трафик, позиции по ключевым запросам, CTR в поиске.

---

### 6. Marketing Agent (`05_MARKETING_AGENT`)
**Приоритет:** P2 — средний  
**Ответственность:** создание маркетинговых кампаний, офферов, промо-материалов, управление воронками продаж, A/B тестирование, сезонные акции.  
**Входы:** аналитика (Analytics Agent), каталог (Product Database), тренды рынка.  
**Выходы:** рекламные тексты, офферы, контент-план, лэндинги.  
**KPI:** ROAS, CPL, CTR рекламных материалов, % роста аудитории.

---

### 7. Analytics Agent (`06_ANALYTICS_AGENT`)
**Приоритет:** P2 — средний  
**Ответственность:** сбор и обработка данных со всех агентов, построение дашбордов, выявление аномалий, прогнозирование спроса, еженедельные отчёты.  
**Входы:** данные продаж, трафика, рекламы, остатков.  
**Выходы:** дашборды, отчёты, алерты об отклонениях, прогнозы.  
**KPI:** coverage метриками, точность прогнозов, скорость формирования отчётов.

---

### 8. Video Agent (`07_VIDEO_AGENT`)
**Приоритет:** P3 — плановый  
**Ответственность:** создание видеоконтента для соцсетей и маркетплейсов: сценарии, обзоры товаров, обучающие ролики, Reels/Shorts, монтажные инструкции.  
**Входы:** карточки товаров, маркетинговые задачи, тренды платформ.  
**Выходы:** сценарии, готовый видеоконтент, описания к видео.  
**KPI:** охваты, вовлечённость, конверсия с видео в продажи.

## ПРИНЦИПЫ РАБОТЫ ФАБРИКИ

1. **Automation First** — любой повторяющийся процесс автоматизируется.
2. **Data-Driven** — каждое решение опирается на данные из Analytics Agent.
3. **Single Source of Truth** — Product Database является единым источником данных о товарах.
4. **Async by Default** — агенты работают асинхронно, результаты передаются через очередь задач.
5. **Owner-in-the-Loop** — владелец получает недельные дайджесты и участвует только в стратегических решениях.
