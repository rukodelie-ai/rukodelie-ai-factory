### Замеченные особенности данных

**Бренд:** у 18 из 50 товаров бренд явно не указан или указана только страна производства. Требует ручного заполнения или уточнения у владельца.

**Единицы измерения остатка:** товары категорий "Лента" и "Бахрома" имеют остаток в **метрах**, а не штуках. В CSV это не размечено — нужно добавить поле `unit` (шт / метр).

**Два формата артикулов:** у ряда товаров есть внешний артикул (например, `00-00003382`) и внутренний RD-код (`20/44/2-`). В CSV сохранён один из двух. Рекомендуется добавить поле `article_rd`.

**Товары с остатком = 1:** 7 товаров из 50 имеют остаток 1 шт. — требуют первоочередной проверки актуальности.

## 4. РЕКОМЕНДАЦИИ ПО МАССОВОМУ ИМПОРТУ ВСЕГО КАТАЛОГА

### Оценка объёма

| Параметр | Значение |
|----------|----------|
| URL в sitemap (видимые) | 300+ |
| Реальная оценка каталога | 500–1500 SKU |
| Время на полный сбор (одиночные запросы) | 3–5 часов |
| Время на полный сбор (параллельно, 10 потоков) | 30–60 минут |

### Доработки перед массовым импортом

**Шаг 1 — Расширить схему CSV:**
```
id, name, category, brand, article, article_rd, price_kzt, stock, unit,
weight_g, length_m, composition, country, description_short,
description_full, photo_urls, keywords, url, date_parsed
```

**Шаг 2 — Выгрузить полный sitemap:**  
Текущий sitemap при просмотре обрезается. Нужно скачать полный XML-файл и извлечь все URL.

**Шаг 3 — Приоритизировать категории:**  
Начинать с товаров высокого спроса (пряжа, нитки, молнии, фурнитура), а не в порядке sitemap.

**Шаг 4 — Добавить поле `unit`:**  
Для тканей, лент и бахромы единица = метр, для остальных = шт.

**Шаг 5 — Генерация SEO-контента:**  
После сбора технических данных — AI Project Manager генерирует SEO-описания и ключевые слова по шаблону из `10_PROMPTS`.

**Шаг 6 — Верификация владельцем:**  
Выборочная проверка 10–20 позиций на актуальность цен и остатков перед использованием в AI Seller.

### Приоритетный порядок категорий для следующего импорта

| Приоритет | Категория | Обоснование |
|-----------|-----------|-------------|
| P0 | Пряжа (YarnArt, Alize, Drops) | Топ продаж, высокий средний чек |
| P0 | Нитки швейные | Высокая оборачиваемость |
| P1 | Молнии (рулонные и штучные) | Большой ассортимент, стабильный спрос |
| P1 | Пуговицы и фурнитура | Широкий ассортимент |
| P2 | Ткань прикладная | Сложная номенклатура (требует цвет, ширина, состав) |
| P2 | SWAROVSKI | Специфическая аудитория |
| P3 | Украшения, броши | Низкий остаток у многих позиций |

## 5. СЛЕДУЮЩИЙ ШАГ

**Действие:** Владелец проверяет 10 позиций из `catalog_raw.csv` на актуальность цен и остатков.

**Что проверить:**
- Совпадает ли цена в CSV с реальной ценой на сайте
- Совпадает ли остаток с фактическим наличием на складе
- Корректны ли артикулы для поиска в 1С

**После подтверждения:** запустить полный сбор каталога по категориям P0 (пряжа, нитки).
