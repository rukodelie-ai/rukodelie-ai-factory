## 2. Ключевые решения сессии

### Решение #13: RAM для истории диалогов (FROZEN)

**Вопрос:** RAM dict vs SQLite для хранения истории диалогов Claude API.

**Анализ:**

| Критерий | RAM (dict) | SQLite |
|----------|-----------|--------|
| Строки кода | ~5 | ~70 (+57%) |
| Зависимости | 0 | +1 (aiosqlite) |
| Скорость | O(1) мкс | O(log n) мс |
| Персистентность | нет | есть |
| Для MVP (<100 сессий) | достаточно | избыточно |

**Решение: RAM.** При рестарте бота история сбрасывается — допустимо для MVP. SQLite добавляет 57% кода при нулевом пользовательском преимуществе на целевом масштабе.

**Когда пересматривать:** когда бот достигнет 100+ одновременных сессий или потребуется аналитика диалогов.
